# Examples｜品牌语气转译示例

以下示例展示 Brand Voice Translator 如何根据产品信息、语气类型、使用场景和禁用表达，生成可用于真实商业场景的品牌文案。所有示例均为原创表达，不指向、不模仿任何真实品牌。

## Example 1：身体油文案转译

### Input

原始文案：

这是一款身体油，含有 60% 油相精华，质地轻盈不黏腻，可以滋润肌肤，同时带有温暖花香，适合睡前使用。

产品类型：

身体油

目标语气类型：

东方感官叙事型

文案风格示意：

睡前，把香气轻轻推开。希望有温柔、感官、睡前仪式感，但不要欲望化。

使用场景：

详情页首屏文案

希望语气：

温柔、感官、睡前仪式感

禁用表达：

引诱、性感、撩人、精致人生

### Output

#### 1. 语气判断

- 情绪温度：温柔、安静、偏夜间护理感。
- 句子节奏：慢节奏，有停顿和留白，适合详情页首屏。
- 词汇风格：围绕“睡前、肌肤、花香、轻盈、柔润”展开。
- 商业直接度：中低，不强推购买，先建立使用场景。
- 适合的视觉 / 生活方式联想：浴后、床头、暖光、睡衣、夜间身体护理。
- 需要避免的表达：避免欲望化、暧昧化和过度文艺化，不使用“引诱”“性感”“撩人”等词。

#### 2. 转译版本 A｜品牌详情页版

睡前，把温暖花香轻轻推开。  
60% 油相精华带来柔润触感，质地轻盈，推开后不黏腻。它适合在沐浴后的夜晚使用，让肌肤被一层细腻油光包裹，也让身体慢慢进入安静的休息节奏。

#### 3. 转译版本 B｜社交媒体版

很适合睡前用的一款身体油。  
花香是暖的，油感很轻，推开之后不会黏在睡衣上。洗完澡后花几分钟按一按，肌肤会有柔润的光泽，整个人也像被夜晚慢慢安抚下来。

#### 4. 转译版本 C｜短句广告感

- 睡前，把花香轻轻推开。
- 轻盈油感，柔润到夜里。
- 给肌肤一层不黏腻的温柔。
- 沐浴后，身体慢慢安静下来。
- 温暖花香，留在睡前几分钟。
- 不是厚重滋润，是刚刚好的柔润感。

#### 5. 表达优化建议

原文中的“60% 油相精华”“轻盈不黏腻”“滋润肌肤”“温暖花香”被保留，并转译成更适合详情页首屏的身体感受和睡前场景。文案规避了“引诱、性感、撩人”等欲望化词汇，也避免用“精致人生”这类泛化表达替代产品信息。若要更偏转化，可以增加肤感对比和使用步骤；若要更偏品牌，可以强化夜间护理仪式；若要更偏社媒，可以继续压短句子、增加真实体验感。

## Example 2：饰品上新文案转译

### Input

原始文案：

这是一组银色花朵和贝壳元素的饰品，适合夏天、出游、聚会佩戴，有光泽感和造型感，但不会太隆重。

产品类型：

饰品

目标语气类型：

都市明媚饰品型

使用场景：

小红书上新文案

希望语气：

明媚、轻盈、适合夏日出游和聚会

禁用表达：

高级感、精致女人、轻奢、贵气

### Output

#### 1. 语气判断

- 情绪温度：明亮、轻松、有夏日空气感。
- 句子节奏：短中句结合，适合小红书阅读和收藏。
- 词汇风格：光泽、银色、花朵、贝壳、出游、聚会、造型细节。
- 商业直接度：中等，既要有上新信息，也要有佩戴场景。
- 适合的视觉 / 生活方式联想：白裙、吊带、海边、城市露台、夏日晚餐。
- 需要避免的表达：避免“高级感”“轻奢”“贵气”等空泛词，也不把女性塑造成单一审美模板。

#### 2. 转译版本 A｜品牌详情页版

银色花朵与贝壳线条，像把夏天的光留在耳边。  
这一组饰品保留了足够的光泽和造型感，却不会显得过分隆重。适合出游、聚会，也适合给日常穿搭加上一点更明亮的细节。

#### 3. 转译版本 B｜社交媒体版

夏天真的很适合戴一点银色。  
花朵和贝壳元素都很轻，不会太甜，也不会太隆重。配白裙、吊带或者简单 T 恤都好看，出游拍照有光泽，晚上聚会也够有造型感。

#### 4. 转译版本 C｜短句广告感

- 把夏天的光戴在耳边。
- 银色花朵，轻轻点亮一身搭配。
- 不隆重，但很有存在感。
- 适合出游，也适合晚一点的聚会。
- 贝壳与花朵，都是夏天的细节。
- 简单穿搭，也需要一点明亮的光。

#### 5. 表达优化建议

原文的“银色花朵和贝壳元素”“适合夏天、出游、聚会”“有光泽感和造型感”被转化为更具体的佩戴画面。文案规避了“高级感、精致女人、轻奢、贵气”等容易空泛或过时的表达。若要更偏转化，可以补充材质、尺寸、佩戴重量；若要更偏品牌，可以强化系列主题；若要更偏社媒，可以加入更生活化的穿搭组合。

## Example 3：茶礼盒文案转译

### Input

原始文案：

这是一款茶礼盒，包含白桃茉莉花茶、海盐白茶、亚麻茶巾和祝福卡，适合送朋友、同事和客户。

产品类型：

茶礼盒

目标语气类型：

新中式雅致型

使用场景：

礼盒详情页 / 品牌手册

希望语气：

雅致、节制、有礼意

禁用表达：

高端、尊贵、古法、传承

### Output

#### 1. 语气判断

- 情绪温度：温和、克制、有心意但不过分煽情。
- 句子节奏：偏书面，留有余地，适合礼盒详情页和品牌手册。
- 词汇风格：茶香、礼意、风味、祝福、关系、片刻、坐下来。
- 商业直接度：中低，重点是礼盒内容和送礼关系。
- 适合的视觉 / 生活方式联想：茶席、亚麻材质、祝福卡、浅色包装、节令拜访。
- 需要避免的表达：避免“高端、尊贵、古法、传承”等容易显得陈旧或夸大的词。

#### 2. 转译版本 A｜品牌详情页版

一份适合慢慢打开的茶礼。  
礼盒内含白桃茉莉花茶、海盐白茶、亚麻茶巾与祝福卡，把清甜、微咸、柔软与心意放在同一份礼物里。适合送给朋友、同事或客户，在不必多言的关系里，留下一段刚刚好的茶香。

#### 3. 转译版本 B｜社交媒体版

这盒茶礼很适合送给“不想送得太重，但也不想太随便”的关系。  
白桃茉莉是清甜的，海盐白茶更轻一点，里面还配了亚麻茶巾和祝福卡。送朋友、同事、客户都合适，礼貌，但不生硬。

#### 4. 转译版本 C｜短句广告感

- 一盒茶，送一段刚刚好的心意。
- 清甜与微咸，都放进这份茶礼里。
- 礼不必重，心意要恰好。
- 给朋友，也给值得认真相待的人。
- 茶香之外，还有一张写下祝福的卡。
- 适合拜访，也适合感谢。

#### 5. 表达优化建议

原文中的礼盒内容被完整保留，并转译为“风味组合 + 礼物关系 + 使用场景”的表达。文案避免了“高端、尊贵、古法、传承”等容易造成距离感或陈词滥调的词。若要更偏转化，可以增加适合节日、价格带或包装规格；若要更偏品牌，可以补充系列主题和视觉关键词；若要更偏社媒，可以加入“送礼对象选择”的口语化建议。

## Example 4：笔记本文案转译

### Input

原始文案：

这是一款为女性日常工作和生活设计的笔记本，可以记录周期、情绪、工作计划和日常灵感。

产品类型：

笔记本

目标语气类型：

女性日常陪伴型

使用场景：

小红书内容 / 产品介绍

希望语气：

真实、温柔、陪伴感、生活化

禁用表达：

自律女性、精英女性、掌控人生

### Output

#### 1. 语气判断

- 情绪温度：温柔、真实、不过度励志。
- 句子节奏：自然口语与轻度产品说明结合。
- 词汇风格：记录、周期、情绪、计划、灵感、日常、陪伴。
- 商业直接度：中等，需要清楚说明记录功能，但不要变成工具型清单。
- 适合的视觉 / 生活方式联想：书桌、通勤包、床头、咖啡店、月度计划页。
- 需要避免的表达：避免“自律女性、精英女性、掌控人生”等给用户压力的表达。

#### 2. 转译版本 A｜品牌详情页版

一本为女性日常节奏设计的笔记本。  
它可以记录工作计划，也可以留下周期、情绪和突然出现的灵感。不是为了让每一天都被安排得完美，而是给生活里那些细小但重要的变化，一个可以被看见的位置。

#### 3. 转译版本 B｜社交媒体版

有些事情不一定要立刻解决，但可以先写下来。  
这本笔记本可以记工作计划，也可以记周期、情绪和日常灵感。忙的时候用它理一理待办，累的时候用它放一放心情。它更像一个安静的陪伴，而不是催你变得更好的工具。

#### 4. 转译版本 C｜短句广告感

- 记录计划，也记录身体和心情。
- 给日常一个可以停下来的地方。
- 不催促，只陪你慢慢理清。
- 工作、周期、情绪和灵感，都可以被写下。
- 一本更懂日常节奏的笔记本。
- 把细小的变化，认真留住。

#### 5. 表达优化建议

原文中的功能点“周期、情绪、工作计划、日常灵感”被保留，但表达从工具卖点转向女性真实日常和陪伴感。文案规避了“自律女性、精英女性、掌控人生”等容易制造压力的表达。若要更偏转化，可以补充内页设计、纸张、尺寸和栏目结构；若要更偏品牌，可以强化“女性日常节奏”的核心概念；若要更偏社媒，可以增加使用场景照片建议和标题版本。
