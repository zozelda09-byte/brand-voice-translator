# Brand Voice Translator｜品牌语气转译 Skill

## What is it?

Brand Voice Translator 是一个用于品牌文案转译的 ChatGPT Skill。它可以将普通产品信息、基础卖点或初稿文案，转化为不同语气类型下更适合品牌定位、内容平台和消费语境的表达。

这个 Skill 的重点不是把文案写得更“高级”，而是帮助品牌判断：在某个产品、平台和场景下，文案应该以什么样的情绪温度、句子节奏、词汇风格和商业直接度出现。

请注意：

- 它不是品牌模仿工具。
- 它不会复制任何真实品牌的语言风格。
- 它通过抽象语气类型、使用场景和风格示意来生成文案。
- 它适合品牌策划、内容策划、电商详情页、小红书内容、社交媒体文案、产品上新文案等场景。

## Who is it for?

适合以下角色和工作流：

- 品牌策划
- 内容策划
- 电商文案
- 小红书运营
- 独立品牌主理人
- 自由职业文案
- AI content workflow builders

## What can it do?

这个 Skill 可以输出：

- 品牌语气判断
- 品牌详情页文案
- 社交媒体版本
- 短句广告语
- 标题 / 封面文案
- 表达优化建议
- 禁用词规避
- 语气一致性建议

## Input Fields

建议输入以下字段：

- 原始文案 / 产品信息
- 产品类型
- 目标语气类型
- 文案风格示意
- 使用场景
- 希望语气
- 禁用表达

## Output Structure

默认输出结构：

1. 语气判断
2. 转译版本 A｜品牌详情页版
3. 转译版本 B｜社交媒体版
4. 转译版本 C｜短句广告感
5. 表达优化建议

## Brand Safety

这个 Skill 遵循品牌安全原则：

- 不模仿真实品牌。
- 不复刻品牌专有表达。
- 不输出真实品牌 slogan。
- 不使用可能造成商标或公关风险的表达。
- 真实品牌只能作为内部理解参考，不能作为公开输出标签。

如果输入中出现真实品牌名，应将其转译为抽象语气类型，例如“理性克制编辑型”“东方感官叙事型”“轻快社媒网感型”等，而不是输出“某某式文案”。

## Quick Example

### Input

原始文案 / 产品信息：

这是一款身体油，质地轻盈不黏腻，含有植物油脂，可以滋润肌肤，带有温暖花香，适合睡前使用。

产品类型：

身体油

目标语气类型：

东方感官叙事型

使用场景：

详情页首屏文案

禁用表达：

引诱、性感、撩人、精致人生

### Output

#### 1. 语气判断

这组文案适合以“睡前身体护理”作为切入点，用温柔、感官、慢节奏的语言建立使用场景。表达上应避免欲望化和过度销售感，把重点放在肌肤触感、香气停留和夜间仪式感上。

#### 2. 转译版本 A｜品牌详情页版

睡前，把一层温暖花香轻轻推开。  
轻盈油感贴合肌肤，不厚重，也不急着留下痕迹。植物油脂带来柔润触感，让身体在夜晚慢慢回到松弛的节奏里。

#### 3. 转译版本 B｜社交媒体版

很适合放在床头的一瓶身体油。  
洗完澡后按开一点，花香是暖的，油感很轻，推开后不会黏在睡衣上。像给睡前留出几分钟，把身体慢慢安静下来。

#### 4. 转译版本 C｜短句广告感

- 把夜晚，涂成温柔的花香。
- 轻盈油感，睡前刚刚好。
- 给身体一段慢下来的时间。
- 花香停在肌肤上，也停在夜里。
- 不厚重的滋润，不打扰的香气。

#### 5. 表达优化建议

原始信息中的“轻盈不黏腻”“滋润肌肤”“温暖花香”被保留，并转化为睡前使用场景中的触感和情绪体验。文案规避了“引诱”“性感”“撩人”等容易欲望化的表达，也避免了“精致人生”这类空泛词。
