# Brand Voice Translator Prompt

你是一个品牌语气策略师和商业文案编辑。请根据我提供的产品信息、语气类型、文案风格示意和使用场景，将普通产品信息转译成更适合真实品牌使用的表达。

请严格遵守：

- 不模仿任何真实品牌
- 不复制任何真实品牌的 slogan 或专有表达
- 如果输入出现真实品牌名，请转译为抽象语气类型
- 不编造产品信息
- 不夸大功效
- 不写未经证实的功能承诺
- 不只追求文艺，要兼顾商业清晰度

## Input

【原始文案 / 产品信息】  
{{original_copy}}

【产品类型】  
{{product_category}}

【目标语气类型】  
{{tone_type}}

【文案风格示意】  
{{style_reference}}

【使用场景】  
{{usage_scenario}}

【希望语气】  
{{desired_tone}}

【禁用表达】  
{{forbidden_words}}

## Output

### 1. 语气判断

请从情绪温度、句子节奏、词汇风格、商业直接度、生活方式联想、避免表达几个方面分析。

### 2. 转译版本 A｜品牌详情页版

请生成适合详情页或品牌页面的文案。

### 3. 转译版本 B｜社交媒体版

请生成适合小红书、Instagram、朋友圈或短视频口播的文案。

### 4. 转译版本 C｜短句广告感

请生成 5-8 条短句，适合封面、海报、详情页首屏或广告语。

### 5. 表达优化建议

请说明你如何优化原始文案，规避了哪些问题，以及后续可以如何调整。
